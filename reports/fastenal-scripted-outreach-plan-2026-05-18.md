## What To Say If They Push Back

"We already carry Loctite/other brands."

> That makes sense. We are not asking Fastenal to replace incumbents immediately. The use case is production-size CA adhesive where a customer is consuming by the case and cares about cost per ounce, availability, and replenishment. We can start as customer-driven/direct-ship, then expand only if repeat demand is proven.

"Is this retail glue?"

> No. The main SKU is 16oz, 30 units per case. It is for production, repair benches, MRO, and light manufacturing users.

"Do you have compliance docs?"

> Yes. We can provide SDS/TDS, COO, shelf life, hazmat/GHS summary, label/product photos, case-pack data, and wholesale pricing.

"Do you support EDI?"

> We are preparing the supplier packet now. For initial customer-driven orders we can support manual processing if allowed, and we are evaluating the EDI path if Fastenal requires it for expansion.

"Do you want branch stock?"

> Not necessarily on day one. The cleanest path is customer-driven or direct-ship first, then branch/DC stocking if repeat demand proves it.
